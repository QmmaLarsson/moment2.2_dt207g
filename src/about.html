Om

Denna webbplats erbjuder möjligheten att skapa ett professionellt CV. På sidan "Lägg till nytt jobb" kan man lägga till sina arbetslivserfarenheter, dessa listas sedan på webbplatsens startsida. Den information som kan läggas till om ett jobb är företagsnamn, titel, plats, startdatum och slutdatum. Lägger man inte tll något slutdatum så kommer texten "Pågående" att visas istället.

Webbplatsen använder sig av Fetch API för att hämta data (GET), skicka data (POST) samt radera data (DELETE) från och till en extern webbtjänst. Den externa webbtjänsten är kopplad till en Postgre-databas. Både webbtjänsten och databasen är publicerade på render.com.
